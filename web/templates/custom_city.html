Поддерживаются различные форматы записи координат

Как использовать

1. Выберите формат координат:

  • • Десятичные — обычные числа (61.25, 73.4333)
  • • Градусы-минуты-секунды — раздельные поля
  • • Текстовый ввод — скопируйте координаты как есть

2. Введите координаты интересующего вас места

3. Опционально укажите название города

4. Нажмите кнопку для получения данных

💡 Координаты можно скопировать из Google Maps, Яндекс.Карт или Википедии

🔄 Текстовый ввод поддерживает большинство форматов записи координат
