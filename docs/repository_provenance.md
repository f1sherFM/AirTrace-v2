# Repository Provenance

## Split Origin

`AirTrace-v2` was split from the repository [`Airtrace-RU`](https://github.com/f1sherFM/Airtrace-RU), branch `airtrace-v2`.

- Source repository: `https://github.com/f1sherFM/Airtrace-RU`
- Source branch: `airtrace-v2`
- Source commit: `e1170d8b53c35f6d4f00dccfaa49133cf5d205a6`
- Split date: `2026-04-02`

## Why the Split Happened

The split was performed to make `v2` a standalone product codebase with:

- separate CI and deployment lifecycle
- separate issue tracker and release cadence
- reduced legacy `v1` cognitive load
- cleaner VPS and production operations

## Historical Reference

The original `Airtrace-RU` repository remains the historical reference for:

- pre-split evolution
- legacy `v1` runtime
- early transition history before `v2` became repo-native
